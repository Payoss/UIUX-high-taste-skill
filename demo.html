01 — Skill Collection

Every skill, explained.

Each skill has a specific purpose. Some define aesthetics, others enforce quality. Combined, they produce interfaces that feel designed by humans.

taste-skill

The core anti-slop engine. Defines 3 dials (Variance, Motion, Density) that control every design decision. Bans Inter font, purple AI gradients, centered heroes, 3-column equal card grids. 02 — Key Techniques Applied Here

What you're looking at
right now.

This page uses techniques from multiple skills. Here's what came from where.

01
taste-skill / Leonxlnx
Grain Overlay
SVG noise filter at 2.5% opacity fixed over the entire viewport. Breaks digital flatness without performance cost. pointer-events: none so it never blocks interaction.
02
soft-skill / Leonxlnx
Double-Bezel Cards
Each card has an outer shell (2px padding, subtle border) and an inner core (distinct background + inset highlight). Creates a "machined hardware" depth that flat cards can't achieve.
03
taste-skill / Leonxlnx
Ambient Mesh Gradients
Blurred radial gradients (emerald + blue) floating behind the hero with CSS animation. The "Ethereal Glass" archetype from the taste-skill creative variance engine.
04
taste-skill / Leonxlnx
Staggered Cascade Animation
Cards mount with incremental delays (100ms per card). fadeUp animation with cubic-bezier(0.16, 1, 0.3, 1) easing — the spring-physics feel. Never mount everything at once.
05
soft-skill / Leonxlnx
Serif Accent in Headlines
Mixing Cabinet Grotesk (900 weight sans-serif) with Instrument Serif italic for the word "every other". Creates typographic tension and editorial sophistication.
06
minimalist-skill / Leonxlnx
Mono Metadata Pattern
All labels, navigation, and metadata use Geist Mono at 9-11px, uppercase, with generous letter-spacing. Creates a clear hierarchy: display font for attention, mono for system information.
07
brutalist-skill / Leonxlnx
Swiss Grid Data Layout
The techniques grid itself uses a rigid 2-column structure with visible borders — no rounded corners, no shadows. Oversized numbers at low opacity create structural rhythm. Pure Swiss typographic print influence.
08
mrgoonie-ui-ux-pro-max
Accessibility + Touch Targets
All interactive elements meet 44px minimum touch target. Color contrast ratios verified. Focus states on nav links. Semantic HTML throughout. Priority 1-2 rules from the UX guidelines database.
