Rule 01 — Typography

Inter is banned.

Force Geist, Satoshi, Cabinet Grotesk, or Outfit. Serif banned in dashboards. Monospace for data.

Rule 02 — Color

Max 1 accent. No purple.

Saturation below 80%. The "AI purple gradient" is the #1 tell. Neutral bases + singular accent.

Rule 03 — Layout

No centered heroes.

3-column equal cards = banned. Force split screen, asymmetric grids, bento. CSS Grid over Flexbox math.

Rule 04 — Motion

Spring physics only.

Stiffness 100, damping 20. Staggered cascade reveals. Never mount everything at once. GPU-accelerated only.

content-first.

The Notion / Linear aesthetic. Warm monochrome, desaturated pastel accents, 1px structural borders, zero gradients. Typography does all the work.

T

Typography

Geist Sans body, Lyon Text serif headlines. Off-black #111 text, never pure #000.

C

Colors

Canvas #F7F6F3. Borders #EAEAEA. Desaturated pastels only for accent tags.

L

Layout

Asymmetric bento grids. 1px borders, 8-12px radius. Zero heavy shadows.

B

Buttons

Solid #111 background. 4-6px radius. No pill shapes. No gradients. No box-shadow.

A

Animation

Invisible — present but never distracting. Fade-in, subtle hover shifts. That's it.

S

Spacing

Macro-whitespace between sections. 24-40px internal padding. Let it breathe.
