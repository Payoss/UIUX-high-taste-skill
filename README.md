# UIUX High Taste Skill

> Curated collection of premium UI/UX design skills for Claude Code and AI coding agents.
> Anti-slop, anti-generic, high-taste design systems.

## Install

```bash
npx skills add https://github.com/Payoss/UIUX-high-taste-skill
```

Or copy individual `skills/*/SKILL.md` into your `.claude/skills/` directory.

## Philosophy

**Every pixel matters.** These skills enforce design standards that prevent AI-generated interfaces from looking like AI-generated interfaces. No Inter font, no purple gradients, no 3-column equal cards, no "Elevate your workflow" copy.
